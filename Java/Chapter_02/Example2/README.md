
# Java Example Code 2

## Commands
1. Create folder for class files.
	```
	mkdir bin
	```
2. Compile the code.
	```
	javac -d bin -sourcepath src src\Example2.java
	```
3. Execute the code.
	```
	java -cp bin Example2
	```

## Output
```
This is num: 100
The value of num*2 is 200
Value of num is 100
```
