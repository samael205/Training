
# Hello World Example

## Commands
1. Create folder for class files.
	```
	mkdir bin
	```
2. Compile the code.
	```
	javac -d bin -sourcepath src src\IfStatement.java
	```
3. Execute the code.
	```
	java -cp bin IfStatement
	```

## Output
```
x is less than y.
x is now equal to y.
x is now greater than y.
```
