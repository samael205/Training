
# Java Example Code 2

## Commands
1. Create folder for class files.
	```
	mkdir bin
	```
2. Compile the code.
	```
	javac -d bin -sourcepath src src\ForLoop.java
	```
3. Execute the code.
	```
	java -cp bin ForLoop
	```
